}
window.fn9();

    function heihei(){

varv obj = {
    name:'123'

obj.fn3();

npm小试牛刀

全称为node package manager(node包管理器）

一、npm是一个node包管理和分发工具,已经成了非官方的发布node模块（包）的标准。

npm可以很快的找到特定服务要使用的包，进行下载、安装以及管理已经安装的包。包括：安装、卸载、更新、查看、搜索、发布等

二、npm的包管理命令是使用频率最高的，所以也是我们需要牢牢记住并熟练使用的。

第一步：创建文件夹，初始化npm init，
       其实就是初始化package.json   
第二步：编写自己的组件，利用模块化; 
第三步：如果没有登录npm需要 npm adduser 
      (验证你在 npmjs.org上的账号)；如果登录上了这一步就省略了
第四步：就是发布 npm publish
第五步：就是从线上撤销 npm unpublished name+version

随后会有更新

Git知识

Git技术

git基础知识

本人的github网址https://github.com/gaoshanshansyp/

  • 仓库地址有两种形式https和ssh
    • 安装git，在mac下推荐安装xcode自带git
    • 检测安装git ,终端输入git

  • 第一步：克隆优秀github中的项目

    git clone +仓库地址+（新建名字)

  • 第二步：查看当前仓库的状态

    git status

  • 第三步：修改的文件名称

    git add 文件名称或者全部（.）

  • 第四步：提交到暂存区

    git commit -m 修改日志（就是对项目做了什么操作）

  • 第五步：准备发送到线上仓库

    git remote add origin 仓库路径

  • 第五步：确定发布到线上

    git push -u origin master
