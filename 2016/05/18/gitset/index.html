git基础知识

本人的github网址https://github.com/gaoshanshansyp/

  • 仓库地址有两种形式https和ssh
    • 安装git，在mac下推荐安装xcode自带git
    • 检测安装git ,终端输入git

  • 第一步：克隆优秀github中的项目

    git clone +仓库地址+（新建名字)

  • 第二步：查看当前仓库的状态

    git status

  • 第三步：修改的文件名称

    git add 文件名称或者全部（.）

  • 第四步：提交到暂存区

    git commit -m 修改日志（就是对项目做了什么操作）

  • 第五步：准备发送到线上仓库

    git remote add origin 仓库路径

  • 第五步：确定发布到线上

    git push -u origin master
