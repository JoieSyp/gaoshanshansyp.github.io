npm小试牛刀

全称为node package manager(node包管理器）

一、npm是一个node包管理和分发工具,已经成了非官方的发布node模块（包）的标准。

npm可以很快的找到特定服务要使用的包，进行下载、安装以及管理已经安装的包。包括：安装、卸载、更新、查看、搜索、发布等

二、npm的包管理命令是使用频率最高的，所以也是我们需要牢牢记住并熟练使用的。

第一步：创建文件夹，初始化npm init，
       其实就是初始化package.json   
第二步：编写自己的组件，利用模块化; 
第三步：如果没有登录npm需要 npm adduser 
      (验证你在 npmjs.org上的账号)；如果登录上了这一步就省略了
第四步：就是发布 npm publish
第五步：就是从线上撤销 npm unpublished name+version

随后会有更新
