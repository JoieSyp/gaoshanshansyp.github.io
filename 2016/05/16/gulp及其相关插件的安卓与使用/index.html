  • gulp

gulp.js-基于流的自动化构建工具

gulp的插件库gulp.jspluginregistry

gulp插件的安装

代码如下： 

$ sudo npm install gulp -g

进入项目的的文件夹后初始化开发依赖的的josn文件：

$ npm int

然后设置相关配置后进行一些常用插件的安装，如jsmin，cssmin，jsconcat：

$ npm install gulp-jsmin --save-dev

其他插件的安装和上边一样；

配置gulp任务和参数

在目标文件夹中，创建一个叫gulpfile.js的文件
