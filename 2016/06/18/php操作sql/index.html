php操作sql

  • lamp(m,p)

对于从事前端的我们来说php是一们陌生的语言，但是又特别熟悉，如何想从事全栈，这又是必不可少的技能，我来总结一下自己学到的一点php技巧。

链接数据库

  • 这个函数是用来实例化一个数据库的，它一共有6个参数，但是只有4个经常用到
localhost

这是你所租用数据库的服务器地址(自行查看)

username

这是登录数据库所在服务器在用户名(自行查看)

password

这是登录数据库所在服务器在密码(自行查看)

mysql

这是你的数据库在名称(自行查看)

如何查看数据库是否链接成功

  • $mysqli是我们实例化出来的数据库，名字是你在前面实例化时自己起的。这句话能判断数据库是否链接成功，那么它是什么意思呢？
connect_errno

返回错误编码，0表示没有，其它的都是有错误。

connect_error

返回错误信息，没有错误时为null。

die()

“爹”函数，哈哈！这个函数，是用来打印函数所传参数的值，并且终止此函数后面所有php操作，保护数据安全。

设置读取，写入数据库时的编码格式，现在都用utf8

  • one world!one dream！！！！😂大家都懂的。

书写增删改查语句并执行

  • 增(insert)

在名字叫tableName的表格中添加username为maiime,password为stillAlive的一条数据(username,password是你自己在数据库表格中设置在两列,可以自行设置)

  • 删(delete)

在名字叫tableName的表格中删除username为maiime的数据

  • 改(update)

在名字叫tableName的表格中把username为maiime的数据的username改成stillAlive

  • 查(select)

在名字叫tableName的表格中查找所有数据,星星是通配的意思，就是所有的

  • 别高兴的太早了，这些只是些字符串，真正能执行，还得靠一个函数

需要用这个函数再次执行一下，这个函数是有返回值的，我会在下次更新中详细介绍这个返回值。这是不是说明今天的博客要写完了？别急，还没完。

断开与服务器在链接

使用完数据库一定要及时断开，因为你占用了一条通道，还有就是你永远不知道网络的深处有什么！！！
